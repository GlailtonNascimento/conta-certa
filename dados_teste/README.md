
# 📁 Arquivos de Teste

Esta pasta contém arquivos exemplo para testar o sistema.

## Como testar:

1. Faça upload do arquivo SIAC
2. Faça upload do arquivo da operadora
3. Clique em PROCESSAR

## Arquivos necessários:

Você precisa criar dois arquivos Excel para testar:

### siac_teste.xlsx
| terminal | valor | data |
|----------|-------|------|
| 1 | 12340 | 15/04/2025 |
| 2 | 8900 | 15/04/2025 |
| 3 | 15200 | 15/04/2025 |

### operadora_teste.xlsx
| terminal | valor | data |
|----------|-------|------|
| 1 | 12340 | 15/04/2025 |
| 2 | 8900 | 15/04/2025 |
| 3 | 12000 | 15/04/2025 |

> ⚠️ Neste exemplo, o terminal 3 tem divergência de R$ 3.200
